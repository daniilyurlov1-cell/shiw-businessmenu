# shiw-businessmenu RSG Framework
## Requirements
- [rsg-core]
- [ox-lib]
- [shiw-business] - if want to buy jobs and get rent from players
## Description
This script adds a fully functional configuration menu for the following tasks:
Cashier
Finance
Boss Menu
Storage
Everything is fully customizable and can be modified to suit your needs. It has Russian localization (you can change it in confg.lua) to suit your needs.

## Instructions to incorporate script
- Copy the script into a folder (to choose) from the 'resources' folder.
- Add 'ensure shiw_businessmenu' in the 'Resources.cfg' document
- Configure
